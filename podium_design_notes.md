# Podium Design Observations

## Colour Palette
- Very dark background (near-black, charcoal #1a1a1a or similar)
- White text for primary content
- Muted grey (#888-#aaa) for secondary/metadata text
- Single accent colour (terracotta/rust orange) for highlights only
- White card backgrounds with very subtle borders for the inbox list
- NO multi-colour cards — no red/green/blue/amber per item type

## Inbox UI Design
- Left sidebar: dark background, white text, conversation list
- Each conversation: white/light card, sender name bold, message preview in grey
- Small coloured pill/badge ONLY for status (e.g. "NEW LEAD" in small orange pill)
- Clean conversation thread on the right — white bubbles for customer, dark/blue for business
- Pre-written reply suggestions shown as small pill buttons below the reply box
- No colour noise — the only colour is the accent on status badges

## Layout
- Two-column: left = conversation list, right = open conversation
- Very clean, lots of white space
- Typography: bold sender names, regular weight message preview
- Timestamps in small muted grey
- Action buttons: simple text buttons, not colourful blocks

## Key Takeaways for Blastly Field View
1. Dark background overall, but individual cards/items use near-white or very dark panels
2. Only ONE accent colour for urgency/status — not 5 different colours
3. Pre-written message templates as pill buttons (quick replies)
4. Leads section completely separate from messages section
5. Clean typography hierarchy: bold name, regular preview, muted metadata
6. Minimal borders — just enough to separate items, not decorative
7. The "action" feeling comes from clean layout, not colour
